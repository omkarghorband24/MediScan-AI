About MediScan AI

AI Prescription Analyzer is a smart healthcare web application that extracts medicine information from doctor's prescriptions using OCR technology and provides accurate medicine details in an easy-to-understand format.

About The Project

MediScan AI is designed to simplify the process of reading handwritten and printed medical prescriptions. Patients often find it difficult to understand medicine names, dosage, timing, and duration prescribed by doctors.

This project uses Optical Character Recognition (OCR) to extract text from prescription images. It then compares the extracted medicine names with a medicine database and displays useful information including medicine use, manufacturer, approximate price, side effects, and alternatives.

The system also detects prescription details such as dosage, medicine timing, and treatment duration, helping users better understand their prescriptions.

Our Mission & Vision

Our Mission

To make medical prescriptions easier to understand by providing clear medicine information through Artificial Intelligence and OCR technology.

Our Vision

To build an intelligent healthcare assistant that helps patients, students, and healthcare professionals access medicine information quickly and accurately.

Technologies Used

Python

Backend programming language used to build the AI Prescription Analyzer.

Flask

Lightweight Python framework used to create the web application.

Tesseract OCR

Extracts handwritten and printed text from prescription images.

RapidFuzz

Finds the closest medicine name even if OCR makes small spelling mistakes.

HTML5

Used for building the structure of the web application.

CSS3

Creates a modern, responsive and attractive user interface.

JavaScript

Adds interactivity and improves the user experience.

JSON Database

Stores medicine information including uses, price, side effects and alternatives.

How MediScan AI Works

1

Upload Prescription

Upload a handwritten or printed prescription image.

2

OCR Processing

3

Medicine Detection

4

Display Results

Shows medicine information, dosage, timing and duration.
